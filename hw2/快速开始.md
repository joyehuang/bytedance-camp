# 快速开始指南

## 项目运行步骤

### 1. 安装依赖

```bash
cd hw2
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

服务器将在 `http://localhost:5173/` 启动

### 3. 访问应用

在浏览器中打开 `http://localhost:5173/`

## 功能演示

### 商品列表页 (/)

1. **浏览商品**: 查看商品卡片，包含图片、名称、价格、评分等信息
2. **分类筛选**: 点击左侧分类标签（电子产品、服装、图书等）
3. **价格筛选**: 使用滑块选择价格范围
4. **排序**: 选择排序方式（价格升序、价格降序、销量优先）
5. **搜索**: 在顶部搜索栏输入关键词
6. **分页**: 使用底部分页器浏览更多商品

### 商品详情页 (/product/:id)

1. **查看详情**: 点击商品卡片进入详情页
2. **查看图片**: 点击主图预览，点击缩略图切换
3. **选择规格**: 选择商品规格（颜色、尺码等）
4. **选择数量**: 调整购买数量
5. **加入购物车**: 点击"加入购物车"按钮

### 购物车功能

1. **打开购物车**: 点击右上角购物车图标
2. **查看商品**: 查看已添加的商品
3. **修改数量**: 使用数量选择器调整
4. **删除商品**: 点击删除按钮
5. **清空购物车**: 点击"清空购物车"按钮

## 常见问题

### Q: 如何重置筛选条件？
A: 点击筛选栏底部的"重置筛选"按钮

### Q: 如何返回商品列表？
A: 点击导航栏的"电商平台"标题，或在详情页点击"返回"按钮

### Q: 购物车数据会保存吗？
A: 目前购物车数据存储在 Redux 中，刷新页面会丢失

### Q: 商品数据是真实的吗？
A: 商品数据使用 Mock.js 生成的模拟数据
